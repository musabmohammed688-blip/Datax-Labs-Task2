Task 2: Data Visualization and Storytelling

Objective
To create a visual report using Power BI that translates raw sales data into actionable business insights.

Deliverables Included
* Superstore.csv: The raw dataset used for analysis.
* Untitled.pdf: The final visual dashboard.

Key Insights Identified
1. Top Categories: Technology products drive the highest sum of sales across the business.
2. Sales Trends: Sales show clear cyclical patterns, with significant peaks during the end-of-year holiday quarters.
3. Dashboard Design: Followed clean dashboard design principles by limiting the color palette to prevent clutter and adding clear context labels.
